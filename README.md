# House-Rent-Analysis-Project
This project focuses on performing Exploratory Data Analysis on a house rent dataset to understand rental market trends and patterns. The analysis explores how factors such as city, house type, furnishing status, number of bedrooms (BHK), floor level, area, and tenant preference influence rental prices.
Key steps in this project include:
Data cleaning and handling missing values,
Univariate and bivariate analysis,
Visualization of rent distribution and key features,
Identifying trends and insights across different cities and property types.
Tools Used: Python, Pandas, NumPy, Matplotlib, Seaborn.
